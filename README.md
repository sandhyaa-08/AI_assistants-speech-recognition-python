 Nexa AI Voice Assistant
Nexa AI Voice Assistant is a Python-based voice-controlled assistant that listens to user commands and performs basic system tasks like opening applications. 
It uses speech recognition and text-to-speech technologies to interact with the user in real-time.

🚀 Features

* 🎤 Voice command recognition
* 🔊 Text-to-speech response
* 🧠 Wake word detection ("Hello Nexa")
* 💻 Open system applications (Notepad, Chrome)
* ⏹️ Exit command support

🛠️ Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* playsound
* OS module

📂 Project Structure

AI_spchrecognition.py   # Main Python file
wake up.wav           # Wake sound file
README.md             # Project documentation


⚙️ Installation

1. Clone the repository:
git clone https://github.com/sandhyaa-08/AI_assistants-speech-recognition-python

2. Navigate to the project folder:
cd AI_spchrecognition.py

3. Install required libraries:
pip install SpeechRecognition pyttsx3 playsound

▶️ Usage

1. Run the Python file:
python AI_spchrecognition.py

2. Say the wake word:
Hello Nexa

3. Give commands like:
* "Open Notepad"
* "Open Chrome"
* "Exit"
